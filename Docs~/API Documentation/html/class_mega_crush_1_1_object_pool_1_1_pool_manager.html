Detailed Description

Simple Pooling system for GameObjects. It instantiates multiple instances of predefined prefabs and keeps them in a scene.

The Pool Manager does not need to be instantiated in the scene, all methods are static and can be accessed without a reference, like so:

PoolManager.GetInstance("prefabName");
